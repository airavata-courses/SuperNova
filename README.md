# Technology Stack
* Front End: Angular
* API Gateway: Node.js, Express.js
* Cache Service: Node.js, Express.js, Redis
* Data Retrieval and Plotting - FastAPI, NexDraw, Pyart
* User and Session Management: Spring Boot (Java), H2 SQL
